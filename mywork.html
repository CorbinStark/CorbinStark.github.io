Phantom of Waning Requiem

A 2.5D tactical turn based RPG created using my own Bahamut Game Library. Heavily inspired by games such as Final Fantasy Tactics, Fire Emblem, and Disgaea. Still in early development. All art is my own so development takes a lot longer than I initially anticipated.

Dungeon Crawler VR Game

A VR game made in Unity. Fight your way through dungeons generated from easily editable PNG files. Created for CS410 Virtual Reality with one other teammate.

Defend Your Bounty

A unique tower defence game made for the OGA Spring Game Jam 2019. Won second place overall, first place in gameplay. Build a fort, defend waves of enemies trying to steal your bounty, and survive boss attacks. LINK

Dying is Intended

A short game made as practice for game jams. I gave my myself a limited time span of a week to see how far I could get making a platformer with my own engine. LINK

Bahamut Game Library

A 2D and 3D computer graphics library using OpenGL in C/C++ to render models and handle features such as (but not limited to) reflections, instanced particles, lighting, and cascaded shadowmapping. Includes a custom-written column-major linear algebra module.

Gaffney Orthotics Capstone Project

An Orthotics program for editing and creating 3D models of braces and other othorthes to fit the patient. Created in a team of eight for my Portland State University capstone project. Uses react as a frontend and C++ compiled to web assembly via Emscripten for the backend.







About Me

I work with Software Development for games, game engines, fullstack web dev, and more. I've previously graduated with a Bachelor in Computer Science from Portland State University. I make games as a hobby but I've loved all types of programming since I was young. Game development is just the most fun type.

I thrive working in a team and I enjoy having a variety of tasks and projects. I am skilled at debugging and optimizing code, and I love to learn new things. I began programming when I was 10, but didn't get serious about it until I became the president of my high school's coding club. There I made simple 2D games in Java, before eventually moving to 3D development in C++ with my own custom engine I wrote using OpenGL. Recently I've become interested in modern ray tracing technology, so I am currently self-studying that. I wrote a simple demo of a ray-marched scene in a GLSL fragment shader as my first project, and I plan to learn Vulkan next in order to use their ray tracing tools.
